## Описание раит-апа

Каждая задача полуфинала представлена в отдельной папке и содержит два файла:  

1. **Вопрос (PDF)** — полная формулировка задачи.  
2. **Решение (PY)** — готовый код решения.  

Каждая задача разделена на три раздела:  

1. **Подробное объяснение вопроса** — разбор всех условий и требований задачи.  
2. **Теоретическое объяснение решения** — пошаговое объяснение логики и методов, необходимых для решения.  
3. **Подробный разбор кода решения** — объяснение каждой строки кода и используемых конструкций.  

Такой подход позволяет полностью понять как саму задачу, так и методы её решения на практике.  

> ⚠️ **Примечание:** Этот раит-ап является **неофициальным**, материалы предоставлены для учебных целей.

## Задачи

### Задача A. Стабильна ли строка?
Строка из букв M, T и C считается стабильной, если при любой замене букв на цифры, без ведущих нулей, цифровой корень числа остаётся одинаковым.  
**Входные данные:** стандартный ввод  
**Выходные данные:** стандартный вывод  

### Задача B. Детализация и вирус
Проверка корректности детализации телефонных разговоров, которая могла быть перемешана вирусом. Определение суммарного времени соединений или выявление невозможности корректной детализации.  
**Входные данные:** стандартный ввод  
**Выходные данные:** стандартный вывод  

### Задача C. Отражения сигнала
Моделирование отражений сигнала 5G в тоннеле с окружностью. Определение количества отражений до попадания сигнала в приёмник.  
**Входные данные:** стандартный ввод  
**Выходные данные:** стандартный вывод  

### Задача D. Просто квадрат
По трёхзначному префиксу найти номера телефонов, которые при наборе в международном формате являются простым числом, а в локальном — точным квадратом целого числа.  
**Входные данные:** стандартный ввод  
**Выходные данные:** стандартный вывод  

### Задача E. Самокаты и пробки
Найти количество наборов вершин для минимальной стоимости добавления самокатов на загруженные улицы города.  
**Входные данные:** стандартный ввод  
**Выходные данные:** стандартный вывод
